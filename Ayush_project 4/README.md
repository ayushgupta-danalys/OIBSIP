# Sentiment Analysis Projects

This repository contains two NLP and Machine Learning based sentiment analysis projects completed using Python, Scikit-learn and Jupyter Notebook.

## Projects Included

### 1. Play Store User Reviews Sentiment Analysis

Objective:
Classify Google Play Store user reviews into Positive, Negative and Neutral sentiments using NLP, TF-IDF and Machine Learning models.

Key Techniques:

* Text Preprocessing
* TF-IDF Feature Engineering
* Naive Bayes
* Logistic Regression
* Support Vector Machine (SVM)
* Data Visualization

Folder:


Play Store User Reviews/


### 2. Twitter Sentiment Analysis

Objective:
Classify tweets into Positive, Negative and Neutral sentiments using NLP and Machine Learning techniques.

Key Techniques:

* Text Cleaning
* TF-IDF Feature Engineering
* Naive Bayes
* Logistic Regression
* Support Vector Machine (SVM)
* Data Visualization

Folder:

Twitter Sentiment/
